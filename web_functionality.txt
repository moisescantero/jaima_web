```# Create a virtual scope:
In terminal -> python -m venv venv
add to .gitignore -> venv/
set venv\Scripts\activate in terminal to activate virtual scope
```

```# Install libraries use:
In terminal -> pip install "name of library" (requests, jinja, etc)
```

```# .env file (For use only flask run in terminal to view the changes of our web)
